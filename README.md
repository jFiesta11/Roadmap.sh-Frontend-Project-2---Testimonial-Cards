# Testimonial Cards

### The goal of this project is to demonstrate positioning and layout in CSS. Each card is designed to help you understand different layout and positioning techniques.

## Features:
* Uses HTML 5 and CSS
* Implements Semantic HTML elements and Basic CSS object positioning
* Uses ```:root{}``` to showcase the use of Global CSS Properties

## How to Run:
This project can run locally within your computer without installation.

* Clone or Download this Repository.
    ```bash 
    git clone https://github.com/jFiesta11/Roadmap.sh-Frontend-Project-2---Testimonial-Cards.git
    ```

* Double Click the `index.html` file to open in your default browser or right click the `index.html` file and click on 'open' or 'open with' and select the browser to open this file with.
